# Aadhvik Travels - Static Website

Drop taxi service website for Aadhvik Travels. Rebuilt as pure static HTML/CSS/JS — no Angular, no framework dependencies.

## GitHub Pages Deployment

1. Push this entire folder to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://yourusername.github.io/repo-name/`
